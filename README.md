# RASPBERRYRECORDER

This series of files is used to turn any Raspberry Pi into a Ï€ that can automatically take pictures and record logs about those pictures. 

## Features
- Takes photos at a 10 second interval
- Maintains logs of photos taken and of any power interruptions

## Getting started
You're going to need:
- Raspberry Pi (any)
- Power
- Raspberry Pi Camera
- This repo

Clone this repository, and then use the following command `make run`
